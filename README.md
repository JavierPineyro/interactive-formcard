## Tech Stack

**Client:** React, Typescript, TailwindCSS

**Test:** Vitest, JSdom, @testing-library/react

## Run Locally

1. Clone the project

```bash
  git clone https://link-to-project
```

2. Go to the project directory

```bash
  cd my-project
```

3. Install dependencies

```bash
  npm install
```

4. Start the server

```bash
  npm run dev
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```

- Interactive cards
